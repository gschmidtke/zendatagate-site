ACC, and beyond.

ZEN — Data Gate connects models, metadata, and registries into one calm, automated flow. Batch model health checks, schedule exports, library analytics, and cloud workflows — all from a single, GUID-aware gateway.

  • Bridge Revit and ACC with a GUID-aware gateway designed for large federated projects.
  • Centralize project registries so sheets, links, worksets, and families can be reasoned about together.
  • Explore batch automation across multiple Revit cloud models using a controlled multi-session engine.
What ZEN explores

One gateway, three core ideas.

GUID-aware gateway

ACC & Revit in one story.

ZEN reads from ACC registries, link matrices, and model lists, then opens the right cloud models with a GUID-driven engine. No more manual “which RVT is where?” detective work.

Batch automation

Health checks & exports from one place.

Schedule model health checks, sheet and schedule exports, and analytics across dozens of models. Designed for overnight batch runs rather than manual daily clicking.

Data discipline

Registries as a source of truth.

Treat registries — sheets, links, worksets, families — as first-class citizens. ZEN treats Excel and the CDE as auditable, reviewable sources for project decisions.

Built for BIM teams who care about their data as much as their drawings. Public launch is not planned; this work remains a quiet prototype while options are explored.
